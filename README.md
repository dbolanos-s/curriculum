## Proyectos

### 🤖 Clasificación de Denuncias de Consumidores con Machine Learning
*Proyecto Académico — ESPOL*

Desarrollo de un modelo de Machine Learning para analizar denuncias reales
de consumidores. El objetivo fue enseñarle a la computadora a reconocer
patrones en los datos para predecir automáticamente el motivo de una denuncia
según ciertos factores. Se aplicaron técnicas de clasificación supervisada,
preprocesamiento de datos y evaluación de modelos.

**Tecnologías:** Python · Scikit-learn · Pandas · Machine Learning

---

### 🌱 Sistema de Riego Inteligente con Estación Meteorológica
*Proyecto Académico — ESPOL*

Diseño e implementación de un sistema automatizado de riego que integra una
estación meteorológica local. Utiliza un microcontrolador ESP32 para leer
sensores de humedad de suelo, precipitación y temperatura, tomando decisiones
de riego en tiempo real para optimizar el uso del agua.

**Tecnologías:** ESP32 · Arduino · IoT · Sensores · Python · Automatización
